# Calculadora CLI - Java

Una calculadora interactiva por línea de comandos escrita en Java puro.

## Características

* Operaciones básicas (suma, resta, multiplicación, división).
* Validación estricta de input: el programa atrapa letras, símbolos o valores nulos sin romperse.
* Prevención de división por cero controlada.
* Bucle de ejecución continuo.
